# 🗺️ Project Roadmap

## Overview

This roadmap outlines my DevOps learning journey throughout the IEEE Young Protégé 2026 mentorship program. The goal is to learn DevOps from the fundamentals while applying every concept to the development and deployment of my Personal Safety Emergency Alert System (SOS).

Instead of learning tools in isolation, each topic will be directly connected to the project to gain practical, hands-on experience.

---

# Objectives

## DevOps

- Build a strong foundation in DevOps concepts.
- Gain practical experience with Linux, Git, Docker, Kubernetes, and Microsoft Azure.
- Understand CI/CD pipelines and automation.
- Learn cloud deployment and monitoring.
- Follow secure development practices.

## Software

- Develop a cloud-hosted MVP of the Personal Safety Emergency Alert System.
- Apply DevOps practices throughout the Software Development Lifecycle.
- Continuously improve the application based on feedback.
