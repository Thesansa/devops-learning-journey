# 📅 Week 1 – DevOps Fundamentals, Linux, Git & GitHub

## Overview

This week focuses on building the foundation for my DevOps journey by understanding the core concepts of DevOps, setting up the development environment, and learning the fundamentals of Linux, Git, and GitHub. These concepts will establish the workflow and tools that will be used throughout the development of the Personal Safety Emergency Alert System.

The knowledge gained this week will also be applied by creating and organizing the project repositories, practicing version control with Git, and preparing the development environment for the weeks ahead.

**Topics Covered**
- Introduction to DevOps
- Linux Fundamentals
- Git Fundamentals
- GitHub
- Development Environment Setup

**Application**
- Set up the DevOps Learning Journey repository
- Set up the SOS Emergency Alert System repository
- Initialize version control using Git
- Prepare the development environment
